# Bounty Claim Smart Contracts

Smart contract prepared for the purpose of claiming EDU tokens bounty awards.

1. Migrate the smart contracts on main network
2. Send the EDU tokens to the smart contract address
3. Set the allowance of each contributors using `changeAllowances` function (bulk transactions)
4. Once 0 ETH is sent to the bounty claiming smart contract (from an address with allowance), EDU tokens will be transferred back
